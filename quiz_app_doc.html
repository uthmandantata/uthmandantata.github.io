Developing this project took some precautioned steps that are as follows:

Steps:

  1. Create a dictionary and put questions and their respective answers as key value pair 
    a_dict = {'''
                    Question
                    1.  What is the maximum possible length of an identifier?
                                                
                    Answers
                    A. central processing unit
                    a) 31 characters
                    b) 63 characters
                    c) 79 characters
                    d) None of the above
                    ''': 'd', '''
                    2. Which one of these is floor division?
                    
                    Answers
                    a) /
                    b) //
                    c) %
                    d) None of the mentioned
                    ''': 'b', '''
                    3. Why are local variable names beginning with an underscore discouraged?
                    
                    Answers
                    a) they are used to indicate a private variables of a class
                    b) they confuse the interpreter
                    c) they are used to indicate global variables
                    d) they slow down execution
                    ''': 'a'}
  2. Separate the dictionary into two different lists to get the keys and their values 
    
                    questions_value = list(a_dict.keys())
                    answers_value = list(a_dict.values())
  3. 3.Create a questions variable and a score variable and give them default values of 0. this will enable the program to start from 0 while performing an itteration or making a conditional expression which(ref No. 4) 
    
                    i = 0
                    score = 0
  4. Create a While loop to use the length of the questions lists. Then create an input variable to save the userâ€™s input. The code below takes the questions list with the parameter i , which during the first loop is 0 ( 0 means first). 
    
                    while i < len(questions_value):
                        answer = input(questions_value[i])
  5. To check if the answers given corresponds with the questions, an if statement is used. This creates a condition whereby if question(key) has the value(answer), it will increase the score by 1 and increases i by 1(i is what the while loop uses as a condition to continue) 
    
                    if answer.lower() == answers_value[i]:
                        score += 1
                    i += 1
  6. When i is equals to the length of the questions list, your score will be converted to percentage and a grade will be given 
    
                    if i == len(questions_value):
                        percentage = (score/4)*100
                        if percentage <= 40:
                            print("Your Grade is D")
                        elif percentage <= 60:
                            print("Your Grade is C")
                        elif percentage <= 80:
                            print("Your Grade is B")
                        elif percentage <= 100:
                            print("Your Grade is A")
                        quit()
